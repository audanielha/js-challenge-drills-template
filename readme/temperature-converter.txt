Challenge: Temperature Converter

Objective: Create a JavaScript program that converts temperatures between Celsius and Fahrenheit.

Instructions:

Declare a variable to store a temperature in Celsius (e.g., let celsius = 25;).

Write a function celsiusToFahrenheit that takes the Celsius temperature as an argument and returns the equivalent Fahrenheit temperature. Use the formula: Fahrenheit = (Celsius * 9/5) + 32.

Declare another variable to store a temperature in Fahrenheit (e.g., let fahrenheit = 77;).

Write a function fahrenheitToCelsius that takes the Fahrenheit temperature as an argument and returns the equivalent Celsius temperature. Use the formula: Celsius = (Fahrenheit - 32) * 5/9.
